# domain-trust-grapher

[https://domain-trust-grapher.surge.sh/](https://domain-trust-grapher.surge.sh/) - just go here

### Install dev dependencies
```
$ npm i
```

### Running dev server
```
$ npm start
```

### Building
```
$ npm run-script clean
$ npm run-script build
```

### Deploy to surge.sh!
```
$ npm install -g surge
$ npm surge dist/
```
